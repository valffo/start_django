Стартовый проект

Реализовать личный TODO список, который должен содержать три пункта: Today, Someday, Fixed. Каждый из пунктов может содержать неограниченное количество задач.
Задачи из первых двух списков можно отмечать как выполненные, после чего они отображаются в списке Fixed. В списке Today отображаются не выполненные задачи с датой выполнения не большей, чем текущая дата. В списке Someday отображаются не выполненные задачи с датой выполнения большей, чем текущая дата. Из любого списка можно удалять задачи.

Этап 1
Реализовать обновление списков задач.

На начальной странице:
область элементов управления (ссылки или кнопки) для перехода между списками Today, Someday, Fixed;
активный список, который содержит столбцы: чекбокс; задачу (краткое содержимое); дату выполнения;
контекстно-зависимая область элементов управления (ссылки или кнопки) для операций над задачами активного списка;

Пример начальной страницы.

Someday   Fixed

Today 02.02




Task
Date
*
test stage 1
31.01
*
sport 15:00
02.02
…






Add task  Fix   Remove

Для списка Fixed возможно только операция удаления.

Этап 2
Реализовать операции под списком задач.

Этап 3
Преобразовать приложение в многопользовательское.

Начальная страница приложения содержит форму логина либо ссылку для регистрации. После успешной регистрации / аутентификации пользователь переходит на страницу со своим TODO списком. После завершения работы со списком задач пользователь выходит на начальную страницу.
